/admin 使用说明

这里是当前 OpenClaw 实例的自助管理页面，用于设备审批、上传文件、下载和删除工作区文件。
返回管理页面
1. 设备审批
首次打开 OpenClaw，或更换浏览器、清理缓存后，页面可能提示需要审批 device。
  1. 先打开当前实例的 OpenClaw 页面，按页面提示输入 token。
  2. 如果出现 device approval 提示，回到 /admin/。
  3. 点击 Refresh Device Cache 查看最新请求。
  4. 点击 Approve My Device 审批最近一次请求。
2. 上传文件
上传的文件会进入实例内的 {{ upload_dir }}，可供 OpenClaw 在对话中读取。
  1. 在 Files 区域选择本地文件。
  2. 点击 Upload 上传。
  3. 上传完成后，页面会显示容器内路径。
  4. 单个文件最大 {{ max_upload_mb }} MB。
3. 下载文件
下载列表展示工作区顶层文件，适合下载 OpenClaw 生成的文档、表格、PPT 和压缩包。
  1. 在 Downloadable Files 区域找到文件。
  2. 点击 Download 下载。
  3. 如果文件名唯一，也可以访问 /admin/files/文件名 直接下载。
  4. 支持类型：{{ download_extensions }}。
文件删除和保护规则
可以删除
用户生成或上传的顶层文件通常会显示 Delete 按钮。删除后文件会从当前实例目录中移除。
受保护文件
核心配置和身份文件不会在这里提供删除按钮。当前保护文件包括：{{ protected_filenames }}。
常见情况
情况 处理方式
OpenClaw 要求审批设备 进入 /admin/，先刷新设备缓存，再点击审批按钮。
上传后 OpenClaw 找不到文件 把页面显示的容器路径发给 OpenClaw，例如 {{ upload_dir }}/文件名。
文件列表没有显示想下载的文件 确认文件在 workspace、workspaces 或 uploads 的顶层目录，且后缀属于支持类型。
直接下载链接提示找不到或文件名不唯一 回到 /admin/ 文件列表，使用页面里的 Download 按钮。
